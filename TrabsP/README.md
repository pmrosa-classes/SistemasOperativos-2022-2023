
# Trabalhos Práticos

Os trabalhos práticos pretendem dotar os alunos de conhecimentos práticos, no contexto da unidade curricular mas não necessariamente falados nas aulas.

Todos os trabalhos devem estar organizados num repositório Github (publico no final do projeto) com toda a documentação associada: o que faz, para que server, como se pode compilar, como se pode colocar a trabalhar, etc. No final o Docente ou alguma outra pessoa da área deve conseguir utilizar a ferramenta realizada pelos alunos de forma autónoma.
